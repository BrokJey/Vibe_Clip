# Инструкция по установке иконок навигации

## Шаги для добавления иконок:

1. Скопируйте следующие файлы из папки `VibeClip_Frontend/images/icons/footer/` в папку `VibeClip_Frontend/app/src/main/res/drawable/`:

   - `home.png` → переименуйте в `ic_home.png`
   - `export.png` → переименуйте в `ic_export.png`
   - `folders.png` → переименуйте в `ic_folders.png`
   - `profile.png` → переименуйте в `ic_profile.png`

2. После копирования файлов пересоберите проект (Rebuild Project).

## Что было реализовано:

✅ Иконки подключены к нижней навигации (Лента, Загрузка, Папки, Профиль)
✅ Активная иконка подсвечивается фиолетовым цветом (#9C88FF)
✅ Неактивные иконки отображаются полупрозрачным белым цветом
✅ Текст под иконками также подсвечивается при выборе
✅ Навигационная панель имеет черный фон в соответствии с темой приложения

## Соответствие иконок экранам:

- `ic_home.png` → Лента (Feed)
- `ic_export.png` → Загрузка (Upload)
- `ic_folders.png` → Папки (Folders)
- `ic_profile.png` → Профиль (Profile)
