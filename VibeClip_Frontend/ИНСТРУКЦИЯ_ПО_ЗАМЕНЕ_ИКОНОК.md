# Инструкция по замене иконок и splash screen

## 📱 Где поместить изображения

### 1. ИКОНКА ПРИЛОЖЕНИЯ (App Icon)

#### Вариант А: Адаптивная иконка (рекомендуется)

Поместите ваши изображения в папку:
```
VibeClip_Frontend/app/src/main/res/drawable/
```

**Файлы для замены:**
- `ic_launcher_background.xml` или `ic_launcher_background.png` - фон иконки
- `ic_launcher_foreground.xml` или `ic_launcher_foreground.png` - передний план (логотип)

**Требования к изображениям:**
- Формат: PNG или WebP (без прозрачности для фона)
- Размер: 1024×1024 пикселей
- Для foreground: важная часть (логотип) должна быть в центре 432×432 пикселя

#### Вариант Б: Статические иконки

Замените файлы в следующих папках (в каждой папке должны быть файлы одинакового размера):

```
VibeClip_Frontend/app/src/main/res/mipmap-mdpi/     → 48×48 пикселей
VibeClip_Frontend/app/src/main/res/mipmap-hdpi/     → 72×72 пикселей  
VibeClip_Frontend/app/src/main/res/mipmap-xhdpi/    → 96×96 пикселей
VibeClip_Frontend/app/src/main/res/mipmap-xxhdpi/   → 144×144 пикселей
VibeClip_Frontend/app/src/main/res/mipmap-xxxhdpi/  → 192×192 пикселей
```

**Файлы для замены в каждой папке:**
- `ic_launcher.webp` - обычная иконка
- `ic_launcher_round.webp` - круглая иконка

---

### 2. SPLASH SCREEN (Заставка при загрузке)

Поместите изображение логотипа в:
```
VibeClip_Frontend/app/src/main/res/drawable/
```

**Название файла:** `splash_logo.png` (или любое другое имя)

**Требования:**
- Формат: PNG (с прозрачностью, если нужно)
- Рекомендуемый размер: 512×512 пикселей (или больше, но квадратное соотношение)

**После добавления файла:**
1. Создайте файл `splash_background.xml` в папке `drawable/`
2. Обновите тему в `values/themes.xml`

---

## 🔧 Рекомендации

1. **Для создания иконок** используйте:
   - Android Asset Studio: https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
   - Или онлайн-генератор: https://www.appicon.co/

2. **Формат файлов:**
   - PNG - для изображений с прозрачностью
   - WebP - для оптимизированных изображений без прозрачности
   - XML - для векторных drawable ресурсов

3. **После замены файлов:**
   - Очистите проект: Build → Clean Project
   - Пересоберите: Build → Rebuild Project
   - Удалите приложение с устройства и установите заново
